# User Manual

1. Connect the device to a PC or Laptop. (Using provided A-to-B Cable)
2. Open Max Patch and follow steps in the message box within the patch or continue
here.
3. Press “ON” button to activate the patch and load all serial ports.
4. Using the drop-down menu choose your connected port.
5. Press “PLAY” to play all tracks at same time.
6. Volume is automatically the first Output selection that it starts with by default (when
sensors are active they will control the volume).
7. Choose from any of the other 4 outputs and make sure they light up in patch.

# Important Guidelines:

• Avoid touching the paper clip
• Avoid touching computer or connecting a power supply to laptop
• Avoid wearing any conductive jewelry
